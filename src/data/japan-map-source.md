# Japan map source

`japan-prefectures.json` was generated from **Global Map Japan 2.2**
published by the Geospatial Information Authority of Japan (GSI).

- Source: https://www.gsi.go.jp/kankyochiri/gm_jpn.html
- Original layer: `polbnda_jpn.shp`
- Processing: dissolved into 47 prefectures, topology-preserving weighted
  simplification at 20%, coordinate precision reduced to 0.0001 degrees.
- Terms: https://www.gsi.go.jp/kikakuchousei/kikakuchousei40182.html

User-facing attribution must remain visible while this map is displayed:
`地球地図日本（国土地理院）を加工して作成`.

## Nationwide forecast cells and elevation

`japan-weather-cells.json` is generated by `pnpm generate:cells`.

- The 47 prefecture polygons above are filled with H3 resolution 5 cells.
- The generated dataset currently contains 1,819 land cells, including Hokkaido,
  Okinawa and remote Pacific islands represented by the source geometry.
- Each cell is sampled at H3 resolution 7 to find a representative high point.
- Elevation comes from official GSI `DEM10B` PNG tiles at zoom 8, with the GSI
  global elevation tile (`DEMGM`) as a fallback where DEM10B has no value.
- Runtime weather lookup uses one coordinate per cell. A mountainous cell uses
  its high point when it is at least 400 m high and 250 m above the cell center;
  otherwise it uses the cell center.
- The selected GSI elevation is sent to the weather API for statistical
  downscaling instead of relying on an implicit elevation lookup.
- The UI keeps all 1,819 cells, but a cold refresh requests 180 geographically
  distributed forecast samples. Remaining air-temperature cells use four nearby
  samples with a standard 0.0065 C/m elevation lapse-rate adjustment. Apparent
  temperature is never lapse-rate adjusted because it already combines air
  temperature, humidity, wind and radiation. Scheduled refreshes also store
  exact-coordinate forecasts for published destinations; other discovery
  points are labelled as interpolated estimates. Visitor requests never call
  the provider, and responses are shared for 24 hours.

This grid is for nationwide weather visualization and discovery. It is not a
catalogue of destinations, stations, trails or safe access points. Those remain
separate records linked by stable IDs.

- GSI elevation tile specification: https://maps.gsi.go.jp/development/demtile.html
- GSI tile list: https://maps.gsi.go.jp/development/ichiran.html#dem
